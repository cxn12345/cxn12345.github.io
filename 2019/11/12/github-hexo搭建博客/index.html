github+hexo搭建博客

自己搭建个人博客进行发表文章记录点滴了。自己也算是摸爬滚打搭建成功，然后自己再重新安装部署一遍，把完整步骤分享给大家

一、搭建GitHub Pages

01注册一个GitHub账号（已注册跳过这一步），并登录
02创建仓库
1点击”new”

2填写的仓库名为“用户名.github.io”，用户名一定要和左边用户名一样（切记），不然后面会出错

03启用Github Pages
1、点击“Settings”进入设置

2、往下拉找到“GitHub Pages”，显示如图这样，说明已开启（一般创建完成之后都会默认自动开启）

04 下载安装Git和node.js
1、安装Node.js和配置好Node.js环境,官网下载https://nodejs.org/en/

打开cmd命令行，成功界面如下

2、安装Git和配置好Git环境，安装成功在电脑桌面上任何位置鼠标右键能够出现如下两个选择

二、HEXO

01 安装HEXO
1、在自己认为合适的地方创建文件夹，然后在此文件夹下右击鼠标，选择“Git Bash here”

2、输入 npm install -g hexo进行安装

然后使用 hexo -v ,出现下面界面说明安装成功

3、输入 hexo init 初始化文件夹（漫长的等待·······）看到最后的“Start blogging with Hexo！”
4、输入 npm install,安装所需组件
5、输入 hexo g ，第一次体验hexo

6、输入hexo s，启动本地服务器，通过访问 http://localhost:4000，正式体验Hexo

02 更换主题
使用命令
1、安装主题（yilia）
找到Hexo文件的位置，进入themes文件夹内，这个文件夹就是用来放hexo主题的位置的。在themes文件夹下右键选择Git Bash Here，输入命令,git clone +你复制的链接。
git clone https://github.com/litten/yilia.git
温馨小提示：
（1）此时你应该注意一下你所在文件路径是否正确，应该是在themes文件夹下的
（2）此时的粘贴快捷键不是ctrl+v,而是Insert。
2、启动主题
找到目录下的_config.yml 文件,打开找到 theme：属性，并设置为yilia
3、更换主题
hexo clean //清除一下缓存
hexo g //生成静态页面
hexo s //开启本地服务器
就大功告成了

01 使用Hexo deploy部署到GitHub

2、安装一个扩展

02 检查SSH Keys设置
1、输入cd ~/.ssh
如果出现入下图情况，说明不存在.ssh

解决方法：输入 $ ssh-keygen -t rsa -C “xxxxxxx@qq.com“(自己邮箱)

2、输入eval “$(ssh-agent -s)”，添加密钥到ssh-agent
3、再输入ssh-add ~/.ssh/id_rsa，添加生成的SSH key到ssh-agent

03 添加SSH Key到Github
1、点击“settings”

2、点击“SSH and GPG keys”,然后点击“New SSH key”新建SSH keys
3、填写key

4、输入ssh -T git@github.com，测试添加ssh是否成功。如果看到Hi后面是你的用户名，就说明成功了（有一个会让你输入 yes）
5、设置你的账号信息

04 部署到github
hexo d -g
使用 xxxxxx.github.io 即可访问自己博客

四、更换头像

1、更改自己博客的头像，使用自己想要使用的头像。
例如我的古天乐，嘻嘻嘻，不多说了。
2、我们打开安装主题的文件夹

3、选择我们要修改头像的主题，然后双击进入。

4、双击我们选择主题下的source文件夹
5、选择source文件夹下的img，上传我们需要的更换的图片。

6、在博客主题目录下面进行一个环境的配置，填写我们的地址，然后更新，就完成了更换头像的步骤
